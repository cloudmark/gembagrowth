# **App Name**: GembaGrowth Hub

## Core Features:

- Personal Profile: Showcase Matthew Galea's professional background, expertise, and contact information.
- Blog Post Management: Allow Matthew to easily create, edit, and publish blog posts using a simple markdown editor and templating system.
- Blog Post Display: Present blog posts in a visually appealing, consistent format.
- Contact Form: Enable potential clients to contact Matthew directly through a submission form.
- LinkedIn Integration: Visually show a badge linking directly to Matthew Galea's LinkedIn profile at https://www.linkedin.com/in/mgalea-gembagrowth/

## Style Guidelines:

- Primary color: Forest green (#386641), symbolizing growth, nature, and the 'Gemba' concept of being close to the source.
- Background color: Off-white (#F2F2F2) creating a clean, professional, and calm look. 
- Accent color: Light earthy yellow (#E6B82A) for highlighting important information and CTAs.
- Headline font: 'Belleza', sans-serif, for a touch of art, design, and personality, that matches the user's professional presentation on LinkedIn. Body font: 'Alegreya', serif, as the site's literary, vintage styling will feel inviting.
- Use minimalist icons that represent Gemba (factory/workspace), and growth (trees/plants).
- Use a clean, responsive layout that adapts well to different screen sizes.
- Include a subtle animation of a growing tree or plant on the homepage to symbolize growth and development. This should be a CSS or SVG animation to ensure it's lightweight.
